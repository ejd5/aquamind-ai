5. Ce que j'en pense vraiment (honnêtement)

🏆 Forces réelles

AQWELIA a un vrai avantage concurrentiel sur 3 axes :

  • IA photo (StripScan + diagnostic) : C'est le vrai différentiateur. Aucun concurrent n'analyse les bandelettes par IA ni ne diagnostique l'état de l'eau par photo. C'est patentable/défendable si bien exécuté.
  • Zéro matériel : Pas besoin d'acheter un capteur à 300-450€. Le smartphone suffit. Friction d'entrée = 0. C'est massif pour l'acquisition B2C.
  • Bridge B2C→B2B : Le propriétaire qui galère → trouve un pro via AQWELIA → le pro utilise AQWELIA Pro. Ce flywheel n'existe chez aucun concurrent.

⚠️ Faiblesses réelles

  • Pas de mesure continue : iopool et ICO mesurent en continu (capteur dans l'eau). AQWELIA dépend de l'utilisateur pour faire un test. C'est un désavantage réel pour la détection précoce des problèmes.
  • Précision IA vs capteur : L'analyse par photo de bandelette est moins précise qu'un capteur calibré. Si l'utilisateur prend une mauvaise photo (ombre, flou), le résultat peut être faux. L'IA doit gérer ça gracieusement.
  • Dépendance NVIDIA NIM : L'IA dépend d'une API externe (NVIDIA). Si NVIDIA change sa politique de prix ou limite les crédits, AQWELIA est vulnérable. Le fallback z-ai est une sécurité mais de qualité potentiellement inférieure.
  • Monétisation non prouvée : Le freemium est généreux (Découverte a scans illimités). Le taux de conversion payant n'est pas encore mesuré. Le pricing Oasis (9,99€/mois) est aligné sur iopool mais sans le hardware.
  • B2B très tôt : Lancer Pro en même temps que B2C dilue les ressources. Pro devrait être une phase 2 après validation du B2C.
  • Care (marketplace) non connecté à un vrai stock/fournisseur : Pour l'instant c'est un catalogue. Pas de vraie logistique e-commerce.

💡 Recommandations stratégiques

  • 1. Prioriser le B2C d'abord : Valider le taux de conversion Découverte→Oasis avant de pousser Pro. Objectif : 5% de conversion payante sur 10 000 inscrits = 500 payants = ~30k€/an MRR.
  • 2. Partenariat IoT : Plutôt que de fabriquer un capteur, s'associer à ICO ou iopool pour intégrer leurs données dans AQWELIA. AQWELIA devient le "cerveau" au-dessus des capteurs.
  • 3. Le Spa 365 est intelligent : 6,99€/mois pour le spa uniquement c'est bien positionné. Le marché du spa croît plus vite que la piscine. Bon prix d'entrée.
  • 4. Le Pro à 49€/mois est trop cher pour un indépendant français : Les piscinistes solo en France gagnent ~2500-3500€/mois. 49€/mois = ~1,5% du CA. C'est raisonnable mais il faut prouver la ROI vite. Considérer un plan à 29€/mois pour démarrer.
  • 5. Le Winter Guardian est un atout saisonnier : En hiver, les concurrents sont silencieux. AQWELIA peut garder l'engagement toute l'année. C'est un différenciateur marketing fort.

8. Conclusion

AQWELIA a un vrai potentiel — mais l'exécution est tout

Le produit a 3 atouts qui n'existent nulle part ailleurs :

  1. StripScan IA — analyser une bandelette par photo est un game-changer pour le grand public
  2. Zéro hardware — friction d'acquisition = 0, contrairement à iopool (299€) ou ICO (450€)
  3. Bridge B2C→B2B — le particulier qui galère devient un lead qualifié pour le pisciniste

Mais les risques sont réels :

  • Si l'IA photo n'est pas assez précise → les utilisateurs retournent aux bandelettes manuelles
  • Si le taux de conversion reste < 2% → le modèle freemium ne dégage pas assez de MRR
  • Si iopool lève 5M€ et inonde le marché → difficile de compet' sur la notoriété

Ma recommandation : Focus 100% sur le B2C en Année 1. Prouver la conversion. Mettre Pro en standby. Le Winter Guardian et le Spa 365 sont des niches à occuper vite. Le Care marketplace peut devenir un vrai revenue stream si bien exécuté (commission 10-15% sur les ventes de produits).
